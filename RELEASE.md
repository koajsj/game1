# 发布记录

## 1.4.0 - 2026-05-09

### 玩法与系统

- 游戏主循环切换为固定时间步长（60Hz）
- 增加对象池（粒子、环形特效）降低内存抖动
- 新增移动端移动摇杆与双操作按钮（冲刺 / 脉冲）
- 新增受击反馈：hit stop、红色边缘压迫、震屏强化
- 新增冲刺闪避、冲刺击退与移动朝向反馈
- 新增动态危险区：重力漩涡 / 风暴区
- 新增穿越危险区的高收益核心连锁
- Boss 技能前加入明显前摇提示
- 敌人行为分层：基础陨石 / 环绕型 / 冲锋型
- Boss 技能轮换：环绕召唤、定向召唤、冲刺压制
- 增加局外成长：生命、磁吸升级与资源结算
- 增加埋点统计：死亡原因、局时长

### 画面与音频

- 背景增加双层星云动态光感
- 增加音频 ducking，总线自动让位关键音效

### 工程化

- 新增 GitHub Actions CI（format/lint/e2e）
- Playwright 冒烟测试改为 `file://` 方式，减少环境依赖

## 1.3.0 - 2026-05-09

- 将 `index.html` 内联 CSS/JS 拆分为 `src/style.css` 与 `src/game.js`
- 增加 `npm` 工具链：`eslint`、`prettier`、`@playwright/test`
- 增加 Playwright 冒烟测试（页面加载、开始游戏、移动端按钮）
- 增加 `.editorconfig` 与 `.gitattributes`，统一编码/换行策略

## 1.2.0 - 2026-05-09

- 优化小屏布局：顶部信息区、统计卡片、Boss 面板、弹窗文案
- 增加触控鲁棒性：`pointercancel` 与全局 `pointerup` 兜底
