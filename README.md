# Star Ring: Neon Surge

一个无需构建、可直接在浏览器运行的 Canvas 太空生存动作游戏。

## 在线仓库

- GitHub: https://github.com/koajsj/game1
- 主分支: `main`

## 快速开始

```bash
git clone https://github.com/koajsj/game1.git
cd game1
npm install
```

直接打开 `index.html` 即可游玩。

## 操作方式

- 鼠标移动 / 触摸拖动: 控制飞船
- `WASD` / 方向键: 键盘移动
- `空格` / `Shift`: 闪避冲刺
- `Q`: 脉冲清障
- `P`: 暂停 / 继续
- `R`: 重开

## 主要升级（2026-05-09）

- 固定时间步长循环（60Hz），降低机型差异
- 粒子对象池，减少 GC 抖动
- 移动端移动摇杆 + 双操作按钮（冲刺 / 脉冲）
- 受击反馈增强：短时停顿、红色边缘压迫感、屏幕震动
- 新增冲刺闪避、冲刺穿越与移动方向朝向
- 新增动态危险区：重力漩涡 / 风暴区
- 新增穿越危险区的高收益核心连锁
- Boss 进入攻击前会出现明显前摇提示
- 敌人行为分层（基础/环绕/冲锋）
- Boss 三段技能轮换（环绕召唤/定向召唤/冲刺）
- 局外成长（生命 / 磁吸）与结算资源累积
- 背景星云层与后处理增强
- 音频总线 ducking（音效触发时自动压低背景）
- 运行数据埋点（死亡原因、局时长等）

## 最近优化（2026-05-19）

- 补充 `100svh` / `100dvh` 视口高度回退，改善移动浏览器地址栏伸缩时的画布高度
- 补充 `viewport-fit=cover`，让安全区适配在刘海屏和全面屏移动浏览器中真正生效
- 禁止页面级滚动回弹，减少触摸操作时的误滑干扰
- 小屏下隐藏调试性能浮层，避免遮挡移动端操作区
- 上移移动端摇杆和技能按钮，与底部提示和安全区保持距离
- 超窄屏隐藏底部文字提示，优先保证游戏画面与可点击控件
- 统一 HUD、模式、结算和暂停浮层为中文信息架构，降低移动端理解成本
- 移动端冲刺 / 脉冲按钮会显示冷却倒计时并进入不可点状态，避免误触无反馈
- 视口变化时按比例重映射星空，避免横竖屏切换后背景密度失衡
- 增加低帧率自动降级：持续掉帧时减少星点、弱化高成本光晕和背景层
- 页面切到后台时自动暂停，并在恢复前重置时间累积，避免返回页面后瞬间跳帧
- 增加 `localStorage` 容错和损坏数值兜底，隐私模式或嵌入环境禁用存储时仍可正常游玩
- Playwright 冒烟测试新增画布非空渲染和存储不可用场景

## 质量保障

- `npm run format`
- `npm run format:check`
- `npm run lint`
- `npm run test:e2e`
- `npm run quality`

`test:e2e` / `quality` 需要本机或 CI 环境已经具备 Playwright 浏览器；本仓库不提供安装脚本入口。

已添加 GitHub Actions CI：push/PR 自动执行格式检查、lint、Playwright 冒烟测试。
