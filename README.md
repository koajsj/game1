# 星环冲刺：霓虹风暴

这是一个单页 Canvas 小游戏，直接用浏览器打开 `index.html` 就能游玩。

## 版本说明

当前版本记录见 [RELEASE.md](RELEASE.md)。

## 操作方式

- 鼠标移动 / 触摸拖动：控制飞船
- `WASD` / 方向键：键盘移动
- 空格 / 移动端 `FIRE`：射击
- `Q`：脉冲清障
- `P`：暂停或继续
- `R`：重新开始

## 玩法说明

- 收集绿色星核得分，连击会提高收益。
- 石头、无人机和 Boss 会造成伤害。
- 道具提供护盾、磁铁、减速和加分效果。
- 分数上升后会自动进入更高阶段。
- 背景音乐和音效均由浏览器实时生成，不依赖外部资源。
